# HeyGen 宣传视频素材包

这个目录用于制作“量子知识图谱 · 3D 推演引擎”的约 3 分钟 HeyGen 宣传视频。

## 文件说明

- `narration_zh.txt`：最终中文口播稿。HeyGen 中请直接粘贴这一版，保证口型同步。
- `subtitles_zh.srt`：中文字幕文件。
- `heygen_scene_prompts.md`：HeyGen 头像、画面布局和操作提示词。
- `storyboard/scene_plan.json`：分镜与素材映射。
- `assets/`：项目页面截图素材。
- `exports/quantum_kg_background_sequence.mp4`：给 HeyGen 使用的背景屏幕演示视频。

## 使用方式

1. 打开 HeyGen，创建横屏 16:9 视频。
2. 上传 `exports/quantum_kg_background_sequence.mp4` 作为背景视频。
3. 选择真人感女性虚拟主播，放在左下角圆形框中。
4. 粘贴 `narration_zh.txt` 作为口播脚本。
5. 上传 `subtitles_zh.srt` 或用 HeyGen 自动字幕。
6. 预览口型和字幕同步。
7. 确认无误后再点击最终生成。

## 注意

主播形象应为虚构成年女性，风格可以漂亮、有魅力、科技感强，但不要使用真实公众人物肖像，也不要使用露骨或不适合商业宣传的画面。
